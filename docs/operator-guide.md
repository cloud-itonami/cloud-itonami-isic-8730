# Operator Guide

## First Deployment

1. Register the operator's license, jurisdiction and responsible clinicians/staff.
2. Import historical patient/resident records and counterparties.
3. Run read-only validation of existing records against this blueprint's
   contracts.
4. Configure the Eldercare Governor's hold/escalation policy.
5. Publish a dry-run operation and audit export.

## Minimum Production Controls

- licensed-professional sign-off required before any determination
- finalizing a care plan or incident response always requires a human sign-off
- audit export for every hold, approval and care action
- backup manual process for governor/system outage

## Certification

Certified operators must prove patient/resident-record integrity, governor
independence, evidence-backed reporting and human review for every
high-stakes action.
